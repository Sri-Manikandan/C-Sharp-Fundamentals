# C# Tasks

Collection of C# practice tasks. Each task lives in its own folder as a standalone `dotnet` project.

## Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) (6.0 or later)


## Running a task

From the repo root, run any task with:

```bash
dotnet run --project <task-folder>
```
